
```yaml
生命值：30 HP

```

### 奖励
```yaml

```

### 特点
- ✅ **最常见**：50%刷新率，数量最多
- ✅ **平衡**：攻防血量适中，适合练级
- ✅ **巡逻**：在固定区域巡逻，容易预判

### 战斗策略
```
难度：⭐⭐
推荐等级：1-5级
战术：
1. 保持距离，利用技能远程击杀
2. 近战时注意闪避攻击
3. 可以一次对付2-3只

适合职业：
- 法师：远程魔法轻松击杀
- 战士：近战硬拼，注意血量
- 盗贼：高爆发快速击杀
- 牧师：边打边治疗，稳健
```

```
❌ 血量低于30%时立即撤退
```

## 📊 刷新机制

### 数量管理
```yaml
最大怪物数：15只
最小怪物数：8只
检查间隔：5秒
```

### 刷新触发
```
当前怪物数 < 8只
    ↓
5秒后检测
    ↓
补充到15只
```

### 生成规则
```yaml
距离玩家：至少 300px
距离村庄：至少 200px
生成范围：(100, 100) 到 (1900, 1900)
最大尝试：每只怪物尝试10次位置
```

## 🎮 练级建议

### 1-3级：新手期
```
推荐目标：迷雾幽灵 👻
原因：
- 数量多（50%刷新率）
- 难度低
- 经验稳定

策略：
- 在村庄附近练级
- 一次对付1-2只
- 学会基础战斗技巧
```

### 3-5级：成长期
```
推荐目标：混合击杀
原因：
- 经验需求增加
- 开始有技能了

策略：
- 幽灵为主（70%）
- 蝙蝠为辅（20%）
- 小心暗影之狼（10%）
```

### 5-10级：进阶期
```
推荐目标：暗影之狼 🐺 + 迷雾蝙蝠 🦇
原因：
- 经验效率最高
- 提升战斗技巧

策略：
- 主动寻找暗影之狼
- 用蝙蝠练习走位
- 避免多只狼同时攻击
```

### 10级以上：熟练期
```
推荐目标：全怪混刷
原因：
- 装备和技能都成型了
- 可以高效清怪

策略：
- 快速清理大片区域
- 5秒刷新，永不缺怪
- 经验产出最大化
```

## 💡 进阶技巧

### 技巧1：引怪风筝
```
适用：暗影之狼

步骤：
1. 发现狼后立即后退
2. 保持300px安全距离
3. 边退边释放技能
4. 利用村庄边界限制狼的移动
```

### 技巧2：AOE清怪
```
适用：迷雾幽灵、蝙蝠

步骤：
1. 引诱3-5只怪物聚集
2. 使用AOE技能（如法师的火球术）
3. 一次性获得大量经验
4. 注意血量，及时撤退
```

### 技巧3：村庄卡位
```
适用：所有怪物

原理：
怪物无法进入村庄安全区

步骤：
1. 引怪到村庄边缘
2. 退入村庄内
3. 怪物被推出，无法攻击
4. 安全输出
```

### 技巧4：预判蝙蝠
```
适用：迷雾蝙蝠

技巧：
1. 观察蝙蝠飞行路线
2. 提前释放技能到路径上
3. 让蝙蝠自己"撞上"技能
4. 命中率大幅提升
```
